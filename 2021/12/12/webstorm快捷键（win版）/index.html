webstorm快捷键（win版）

发表于2021-12-12|更新于2023-07-30|编辑器
|阅读量:|评论数:

编辑

Ctrl + Space 基本代码完成 （任何类、 方法或变量名称）
Ctrl + Shift + Enter 完整的语句
Ctrl + P （在方法调用参数） 内的参数信息
Ctrl + Q 快速的文件查找
Ctrl + mouse 在简短的信息的代码
Ctrl + F1 在插入符号显示错误或警告的说明
Alt + Insert 生成代码…
Ctrl + Alt + T 以…（如果……否则，尝试。抓住，等等。）
Ctrl + / 行注释与取消注释
Ctrl + Shift + / 块注释与取消注释
Ctrl + W 选择先后提高代码块
Ctrl + Shift + W 减少当前所选内容到以前的状态
Alt + Q 上下文信息 Alt + enter 键显示意图采取行动和快速修复
Ctrl + Alt + L 格式化代码
Ctrl + Alt + I 自动缩进的行数
Tab / Shift + Tab 缩进/取消缩进选定的行
Ctrl + X , Shift + Delete 剪切到剪贴板的当前行或者所选代码块
Ctrl + C，Ctrl + 插入复制当前行或者所选代码块到剪贴板
Ctrl + V，Shift + 从剪贴板插入粘贴
Ctrl + Shift + V 粘贴从最近缓冲区…
Ctrl + D 复制当前行或者所选代码块
Ctrl + Y 删除线在插入符号
Ctrl + Shift + J 加入行
Ctrl + enter 键分割线
Shift + Enter 开始新行
Ctrl + Shift + U 切换案例在插入符号或所选代码块的词
按住 Ctrl + Shift +] / [选择直到代码块结束/开始
Ctrl + Delete 删除单词结束 Ctrl + 退格键删除单词的开始
Ctrl + 滑鼠 + /-展开/折叠代码块
Ctrl + Shift + 滑鼠 + 全部展开
Ctrl + Shift + 滑鼠-崩溃
Ctrl + F4 关闭活动的编辑器标签

多个插入符号和选择

Alt + 单击添加或插入符号
Shift + Ctrl + Alt + J 选择全部删除按住
Alt + J 选择下一个出现
Alt + Shift + J 取消选择发生按
Esc 取消选择所有事件或插入符号

运行

Alt + Shift + F10/F9 选择配置和运行/调试
Shift + F10/F9 运行/调试
Ctrl + Shift + F10/F9 运行/调试上下文配置从编辑器
Alt + Shift + R 重新运行测试

调试

F8 步骤
F7 一步成
Shift + F7 智能踏入
Shift + F8 踏出
Alt + F9 运行到光标
Alt + F8 计算表达式的值
F9 简历
Ctrl + f8 键切换断点
Ctrl + Shift + F8 查看断点

导航

Ctrl + N 去某个类
Ctrl + Shift + N 转到文件
Ctrl + Alt + 转变 + N 转到符号
Alt + 权利转到下一个编辑器选项卡
Alt + 左转到上一个编辑器选项卡
f12 键回到上一个工具窗口
esc 键转到编辑器 （在工具窗口中）
Shift + Esc 隐藏活动或最后一个活动窗口
Ctrl + Shift + F4 关闭活动运行/消息/查找 /……选项卡
Ctrl + G 转到线
Ctrl + E 最近文件弹出式
Ctrl + Alt + 左/右导航回 /转发
Ctrl + Shift + 退格键导航最后编辑位置
Alt + F1 选择当前文件或符号在任何视图中
Ctrl + B，
Ctrl + 单击转到声明
Ctrl + Alt + B 去实现
Ctrl + Shift + I，打开快速定义查找
Ctrl + Shift + B 去键入宣言
Ctrl + U 转到超级方法/超级类
Alt + 上下转到上一页/下一页方法
Ctrl +] / [移动到代码块结束/启动
Ctrl + f12 键文件结构弹出
Ctrl + H 类型层次结构
Ctrl + Alt + H 调用层次结构
F2 / Shift + F2 下一页/上一页突出显示错误 F4，
Ctrl + Ente 跳转到源
Alt + home 跳转到导航栏
F11 切换书签
Ctrl + F11 切换书签助记键
Ctrl + #[0-9] 去带编号的书签
Shift + F11 显示书签

VCS/当地历史

Alt + 反引号 (‘) ‘VCS’ 快速弹出
Ctrl + K 提交项目到 VCS
Ctrl + T 更新项目从 VCS
Alt + Shift + C 查看最近的更改

搜索/替换

Ctrl + F 查找
F3 查找下一步
Shift + F3 查找上一个
Ctrl + R 替换
Ctrl + Shift + F 在路径中找到
Ctrl + Shift + R 替换路径中

使用搜索

Alt + f7 键 / Ctrl + f7 键查找使用实例 / 找到用法在文件
Ctrl + Shift + f7 键突出显示用法在文件
Ctrl + Alt + F7 显示用法

重构

F5 副本
F6 移动
Alt + delete 安全删除
Shift + F6 重命名
Ctrl + Alt + N 内联可变
Ctrl + Alt + M 提取方法
Ctrl + Alt + V 引进可变
Ctrl + Alt + C 引入的恒定
Ctrl + Alt + P 介绍参数
Ctrl + F6 变化函数签名

一般

Ctrl + Shift +a 查找操作
双击shift 查找所有的（包括文件、命令等）
Alt + #[0-9] 打开相应的工具窗口
Ctrl + Shift + f12 键切换最大化编辑器
Alt + Shift + F 将添加到收藏夹
Alt + Shift + I 检查当前文件与当前配置文件
Ctrl + 反引号 (‘) 快速切换当前计划
Ctrl + Alt + S 打开设置对话框
Ctrl + Tab 切换选项卡和工具窗口
