е®ҡд№ү

еҶ’жіЎжҺ’еәҸпјҲиӢұиҜӯпјҡBubble SortпјүеҸҲз§°дёәжіЎејҸжҺ’еәҸпјҢжҳҜдёҖз§Қз®ҖеҚ•зҡ„жҺ’еәҸз®—жі•гҖӮе®ғйҮҚеӨҚең°иө°и®ҝиҝҮиҰҒжҺ’еәҸзҡ„ж•°еҲ—пјҢдёҖж¬ЎжҜ”иҫғдёӨдёӘе…ғзҙ пјҢеҰӮжһңе®ғд»¬зҡ„йЎәеәҸй”ҷиҜҜе°ұжҠҠе®ғд»¬дәӨжҚўиҝҮжқҘгҖӮиө°и®ҝж•°еҲ—зҡ„е·ҘдҪңжҳҜйҮҚеӨҚең°иҝӣиЎҢзӣҙеҲ°жІЎжңүеҶҚйңҖиҰҒдәӨжҚўпјҢд№ҹе°ұжҳҜиҜҙиҜҘж•°еҲ—е·Із»ҸжҺ’еәҸе®ҢжҲҗгҖӮиҝҷдёӘз®—жі•зҡ„еҗҚеӯ—з”ұжқҘжҳҜеӣ дёәи¶Ҡе°Ҹзҡ„е…ғзҙ дјҡз»Ҹз”ұдәӨжҚўж…ўж…ўвҖңжө®вҖқеҲ°ж•°еҲ—зҡ„йЎ¶з«ҜгҖӮ

еҶ’жіЎжҺ’еәҸд№ӢжүҖд»ҘеҸ«еҶ’жіЎжҺ’еәҸпјҢжҳҜеӣ дёәдҪҝз”Ёиҝҷз§Қз®—жі•иҝӣиЎҢжҺ’еәҸж—¶пјҢж•°жҚ®еҖјдјҡеғҸж°”жіЎдёҖж ·д»Һж•°з»„зҡ„дёҖз«ҜжјӮжө®еҲ°еҸҰдёҖз«ҜгҖӮеҒҮи®ҫжӯЈеңЁе°ҶдёҖз»„ж•°еӯ—жҢүз…§еҚҮеәҸжҺ’еҲ—пјҢиҫғеӨ§зҡ„еҖјдјҡжө®еҠЁеҲ°ж•°з»„зҡ„еҸідҫ§пјҢиҖҢиҫғе°Ҹзҡ„еҖјдјҡжө®еҠЁеҲ°ж•°з»„зҡ„е·Ұдҫ§гҖӮд№ӢжүҖд»Ҙдә§з”ҹиҝҷз§ҚзҺ°иұЎжҳҜеӣ дёәз®—жі•дјҡеӨҡж¬ЎеңЁж•°з»„дёӯз§»еҠЁпјҢжҜ”иҫғзӣёйӮ»зҡ„ж•°жҚ®пјҢиҖҢе·Ұдҫ§еҖјеӨ§дәҺеҸідҫ§еҖјж—¶е°Ҷе®ғд»¬иҝӣиЎҢдә’жҚўгҖӮ

з®—жі•е®һзҺ°йҖ»иҫ‘

еҶ’жіЎжҺ’еәҸз®—жі•зҡ„иҝҗдҪңеҰӮдёӢпјҡ

  2. еҜ№жҜҸдёҖеҜ№зӣёйӮ»е…ғзҙ дҪңеҗҢж ·зҡ„е·ҘдҪңпјҢд»ҺејҖе§Ӣз¬¬дёҖеҜ№еҲ°з»“е°ҫзҡ„жңҖеҗҺдёҖеҜ№гҖӮиҝҷжӯҘеҒҡе®ҢеҗҺпјҢжңҖеҗҺзҡ„е…ғзҙ дјҡжҳҜжңҖеӨ§зҡ„ж•°гҖӮ
  3. й’ҲеҜ№жүҖжңүзҡ„е…ғзҙ йҮҚеӨҚд»ҘдёҠзҡ„жӯҘйӘӨпјҢйҷӨдәҶжңҖеҗҺдёҖдёӘгҖӮ
  4. жҢҒз»ӯжҜҸж¬ЎеҜ№и¶ҠжқҘи¶Ҡе°‘зҡ„е…ғзҙ йҮҚеӨҚдёҠйқўзҡ„жӯҘйӘӨпјҢзӣҙеҲ°жІЎжңүд»»дҪ•дёҖеҜ№ж•°еӯ—йңҖиҰҒжҜ”иҫғгҖӮ
